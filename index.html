Turn any text into a delegated todo with just a touch.




Todoed is a new kind of a ​ seamless todo listing tool, that clips any selected text anywhere on the web browser and (soon), any device, as if that text is an actionable , delegate-able and/or bookmark-able todo item first. Thus, increasing the productivity and accountability of that communicated text.
